CARPETA DE AUDIO AMBIENTE

Esta carpeta está destinada para clips de audio ambiente del juego.

Formatos recomendados:
- .wav para mejor calidad
- .mp3 para menor tamaño de archivo
- .ogg para balance entre calidad y tamaño

Configuración recomendada en Unity:
- Load Type: Streaming (para archivos largos de ambiente)
- Compression Format: Vorbis
- Quality: 70-100%

Para añadir sonidos ambiente:
1. Arrastra los archivos de audio a esta carpeta
2. Selecciona el archivo en el Project window
3. En el Inspector, configura las opciones mencionadas arriba
4. Asigna el clip al AudioManager en la escena
